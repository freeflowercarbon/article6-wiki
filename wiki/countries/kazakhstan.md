# カザフスタン 🇰🇿 — Article 6.2 ホスト国状況

> 最終更新: 2026-05-10

## サマリー
日本JCM（2014年）パートナー。CIS（旧ソ連）諸国で最初に本格的な国内排出権取引制度（KZ-ETS）を導入した国。石油・ガス・石炭産業が主要排出源で、工業省エネ・炭素回収が主なJCMの対象。草原・ステップのNBS可能性も検討されている。

## 基本情報
| 項目 | 内容 |
|------|------|
| NDC目標 | 2030年に1990年比15%削減（条件付き25%） |
| 国内炭素制度 | KZ-ETS（2013〜）アジア最初の強制ETSの一つ |
| 主管省庁 | 生態・天然資源省 |
| JCM状況 | 日本（2014年）✅ |
| CA体制 | 整備中（ETS連動） |
| NBS成熟度 | ⭐⭐ |
