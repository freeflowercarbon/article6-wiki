# Article 6.2 Policy Wiki — Claude操作ガイド

このファイルはClaudeへの指示書です。このwikiをどう使うか・更新するかを記載しています。

## このWikiとは
パリ協定Article 6.2の各国政策状況をトラッキングするMarkdown Wikiです。
LLMが書き・維持します。手動編集は最小限にしてください。

## フォルダ構造
```
article6-wiki/
├── raw/          ← 生データ（ソース文書）。LLMが読む。手動追加もOK
├── wiki/         ← LLMが生成・更新するWiki本体
│   ├── _index.md       全体インデックス（常に最新に保つ）
│   ├── _sources.md     自動収集ソース候補
│   ├── matrix.md       二国間合意マトリクス（最重要）
│   ├── countries/      国別ページ
│   ├── agreements/     協定別ページ
│   └── concepts/       概念解説
└── outputs/      ← Q&Aの出力・レポート（後から参照できるよう保存）
```

## よく使うコマンド（Claudeへの指示例）

### 情報更新
```
「[国名]の最新情報をウェブで調べて、rawとwikiを更新して」
「matrix.mdを最新の状態に更新して」
```

### Q&A（回答をoutputsに保存）
```
「東南アジアでArticle 6.2が最も進んでいる国はどこか、理由付きで教えて。outputs/に保存して」
「シンガポールの買国としての戦略を投資家向けに1ページにまとめて」
「インドネシアとシンガポールのIA交渉の障壁を整理して」
```

### ヘルスチェック
```
「wikiの情報で矛盾・古い情報がないかチェックして」
「更新が必要なページをリストアップして」
```

### 新しいrawデータを追加した後
```
「raw/に新しいファイルを追加した。内容を読んで対応するwikiページを更新して」
```

## Wikiの品質ルール（Claudeが守るべきこと）
1. 情報の確信度が低い場合は `（要確認）` または `（未確認）` と明記
2. ソースは必ず `raw/` ファイルへのバックリンクで示す
3. `最終更新:` 日付を必ず更新する
4. ステータス記号は `matrix.md` の凡例を使う
5. 新しい概念を導入する場合は `wiki/concepts/` にページを作る
