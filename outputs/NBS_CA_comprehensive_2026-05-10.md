## セクション1：NBSプロジェクト承認の制度進捗フェーズ定義
## Section 1: NBS Institutional Progress Phase Framework

Article 6.2でNBSクレジットが国際移転（ITMO）されるまでには、以下の段階を経る必要がある。
NBS credits require the following sequential phases before they can be transferred as ITMOs under Article 6.2.

### JCMフレームワーク（日本主導）での進捗フェーズ
### JCM Framework (Japan-led) Progress Phases

| フェーズ | 内容 | 条件 |
|---------|------|------|
| **Ph.0** | 二国間JCM協定署名 | 政府間合意 |
| **Ph.1** | 合同委員会（JC）設立・稼働 | 定期会合の実施 |
| **Ph.2** | 相手国の国内炭素ガバナンス整備 | 国家登録簿・NFMSの整備 |
| **Ph.3** | 日本がNBSを戦略優先課題に指定 | 方法論開発に向けた政策決定 |
| **Ph.4** | NBS方法論の開発・JC承認 | セクター別方法論（森林・農業等） |
| **Ph.5** | NBSプロジェクト登録・Authorization取得 | 個別プロジェクトの承認 |
| **Ph.6** | NBSクレジット発行・CA適用・移転 | ITMO化完了 |

### シンガポールIAフレームワークでの進捗フェーズ
### Singapore IA Framework Progress Phases

| フェーズ | 内容 | 条件 |
|---------|------|------|
| **Ph.0** | Implementation Agreement（IA）署名 | 政府間二国間合意 |
| **Ph.1** | NEA Eligibility Criteria公表 | ICCの品質基準を規定 |
| **Ph.2** | NBS対象のIAへのプロジェクト公募（RFP） | 特定IAまたは全IA対象 |
| **Ph.3** | プロジェクト選定・長期購入契約（LTPA） | 政府が供給者と契約 |
| **Ph.4** | CA付きITMO移転 | 相手国がNDCからCA控除 |
| **Ph.5** | 企業がカーボン税控除に使用 | NEA認定ICCとして活用 |

## セクション4：シンガポールの入札（RFP）制度詳細
## Section 4: Singapore's Article 6.2 Carbon Credit Procurement (RFP) System

シンガポール政府は、NDC達成のために必要なカーボンクレジットを政府調達するユニークな制度を構築した。
Singapore has established a unique government procurement system for carbon credits needed for its NDC.

### RFP第1回（2024年9月〜2025年2月）

| 項目 | 詳細 |
|------|------|
| 発注者 | NCCS（国家気候変動事務局）+ MTI（貿易産業省） |
| 公募開始 | 2024年9月 |
| 締切 | 2025年2月14日 |
| 対象クレジット種別 | **NBS（自然系）のみ** |
| 対象期間 | 2021年1月1日〜2030年12月31日発生分 |
| 応募者数 | **17社** |
| 応募価格レンジ | **$19〜$41/t-CO2e** |
| 落札数量 | **2,175,000 t-CO2e** |
| 契約期間 | 5年間（2026〜2030年） |
| 契約金額 | 約**S$76百万（USD$60百万）** |
| 落札プロジェクト数 | **4プロジェクト**（ガーナ・ペルー・パラグアイ） |

**評価基準の内訳**:
| 評価項目 | ウェイト |
|---------|--------|
| プロジェクト品質（環境健全性等） | **35%** |
| 価格競争力 | **35%** |
| 企業の能力・実績 | **20%** |
| 実施体制 | **10%** |

> ⚠️ 価格が品質と同等ウェイト → 最安値落札でなく品質重視の調達

ソース: [S&P Global Price Report](https://www.spglobal.com/commodity-insights/en/news-research/latest-news/energy-transition/022525-singapore-article-6-carbon-credit-tender-attracts-19-41mtco2e-price-offers)、[NCCS Award Announcement](https://www.nccs.gov.sg/singapore-will-contract-high-quality-nature-based-carbon-credits-from-four-projects/)、[ESG Today](https://www.esgtoday.com/singapore-signs-deals-for-more-than-2-million-nature-based-carbon-credits/)

### RFP第2回（2025年10月31日〜）

| 項目 | 詳細 |
|------|------|
| 発注者 | MTI（貿易産業省） |
| 公募開始 | **2025年10月31日** |
| 対象クレジット種別 | **NBS + テクノロジー系（両方）** ← 第1回からの変更点 |

ソース: [MTI Singapore Second RFP](https://www.mti.gov.sg/newsroom/singapore-launches-second-request-for-proposals-for-high-quality-carbon-credits/)、[S&P Global Second RFP](https://www.spglobal.com/energy/en/news-research/latest-news/energy-transition/030625-singapore-to-launch-2nd-request-for-proposal-in-2025-to-procure-article-6-carbon-credits)、[Carbon Pulse RFP2](https://carbon-pulse.com/451306/)

### 投資家への示唆
- シンガポールの政府調達がArticle 6.2クレジットの**価格ベンチマーク（$19〜$41）**を形成
- 第2回はテクノロジー系も対象 → JCMプロジェクト（省エネ等）も対象になる可能性
- IA締結済み国（現在：ガーナ・PNG・ルワンダ・ブータン・カンボジア・ペルー・コロンビア・パラグアイ等）に限定
- **東南アジア5カ国（インドネシア・フィリピン等）がIA締結すれば大きな供給機会**

## セクション8：URLインプット方法のガイド
## Section 8: How to Input URLs / PDFs for Wiki Update

このwikiを継続的に更新するために、以下の方法でURLやPDFを共有してください。

### 方法1：このチャットにURLを貼る（最も簡単）
```
以下のURLをクロールして、関連するwikiページを更新して：
https://example.com/policy-update

また、このURLに参照リンクがあれば、クロール先リストへの追加を検討してください。
```
Claudeが自動的に内容を読み、以下を判断します：
- どのwikiページに反映するか
- rawフォルダにどう保存するか
- 参照先URLをクロールリストに追加すべきか

### 方法2：PDFをrawフォルダに直接置く
`article6-wiki/raw/countries/[国名]/`に手動でPDFを保存してから：
```
raw/countries/indonesia/に新しいPDFを追加しました。内容を読んでwikiを更新して。
```

### 方法3：新しいドキュメントのまとめてインプット
```
以下のURLリストを読んで、クロール候補として適切なものを_sources.mdに追加して：
1. https://...
2. https://...
3. https://...
```

### Claudeが自律的に判断すること
- URL内で参照されている他のURLが重要そうであれば、**自律的に`_sources.md`のクロールリストに追加**
- 新しい政策更新であれば対応する`wiki/countries/[国名].md`を更新
- 新しい協定であれば`wiki/agreements/`に新ページを作成
- 既存情報と矛盾する場合は明示してユーザーに確認
